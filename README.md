# Your Energy

**Your Energy** — це інтерактивний веб-застосунок (SPA) для любителів спорту, який допомагає знаходити вправи для тренувань, фільтрувати їх за групами м'язів або обладнанням, переглядати детальну інформацію та отримувати щоденну мотивацію.

Проєкт реалізовано як курсову роботу з використанням чистого JavaScript (Vanilla JS) та збірника Vite.

 **[Посилання на Live Page (GitHub Pages)](https://yan-chan-py.github.io/energy_project/)**

##  Функціонал

Застосунок розроблено відповідно до макету Figma та технічного завдання. Основні можливості:

1.  **Цитата дня**
  - При завантаженні сторінки у сайдбарі відображається мотиваційна цитата та її автор (запит на API виконується один раз).

2.  **Розширені фільтри**
  - Вправи можна фільтрувати за трьома категоріями: **Muscles** (М'язи), **Body parts** (Частини тіла), **Equipment** (Обладнання).
  - Перемикання вкладок динамічно оновлює список категорій.

3.  **Каталог вправ та Пошук** ️
  - Відображення карток вправ з інформацією про рейтинг, спалені калорії та час виконання.
  - Реалізовано пошук вправ за ключовим словом (пошук відбувається після натискання Enter або кліку на кнопку пошуку).
  - Реалізовано пагінацію для зручного перегляду великої кількості карток.

4.  **Модальне вікно (Деталі вправи)**
  - При натисканні кнопки **Start** відкривається модальне вікно з детальним описом вправи, GIF-анімацією виконання та додатковими метаданими.

5.  **Система рейтингу**
  - Користувач може оцінити вправу прямо в модальному вікні (відправка `PATCH` запиту на сервер).

6.  **Підписка**
  - У футері реалізована форма для підписки на новини. Валідація email та відправка `POST` запиту.

## Технології

Проєкт створено без використання фреймворків (No React/Vue/Angular).

-   **Core:** HTML5, CSS3, JavaScript (ES6+ Modules).
-   **Bundler:** [Vite](https://vitejs.dev/) — для швидкої розробки та оптимізованої збірки.
-   **Architecture:** Модульна структура JS, використання `partials/` для HTML-фрагментів.
-   **HTTP Requests:** Axios / Fetch API (для взаємодії з бекендом).
-   **UI Libraries:**
  -   `iziToast` — для сповіщень.
  -   `basicLightbox` (або власна реалізація) — для модальних вікон.
-   **Deploy:** GitHub Pages + GitHub Actions.

##  Встановлення та запуск

Щоб запустити проєкт локально, виконайте наступні кроки:

1.  **Клонуйте репозиторій:**
    ```bash
    git clone [https://github.com/Yan-Chan-py/energy_project.git](https://github.com/Yan-Chan-py/energy_project.git)
    ```

2.  **Перейдіть у папку проєкту:**
    ```bash
    cd energy_project
    ```

3.  **Встановіть залежності:**
    ```bash
    npm install
    ```

4.  **Запустіть режим розробки (Dev Server):**
    ```bash
    npm run dev
    ```
    Відкрийте посилання в терміналі (зазвичай http://localhost:5173).

5.  **Збірка для продакшену:**
    ```bash
    npm run build
    ```
